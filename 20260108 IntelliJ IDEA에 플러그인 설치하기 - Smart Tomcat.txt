[IntelliJ IDEA에 플러그인 설치하기 - Smart Tomcat]


1. Smart Tomcat 플러그인 설치
메인 메뉴 > 파일 > 설정 > 플러그인 : Smart Tomcat 검색 후 설치

이후 서버 연동 작업 필요
우측 상단에 "현재 파일" 클릭 후 "구성 편집" 클릭하면 실행/디버그 구성 창이 뜬다. 왼쪽 상단 + 버튼인 "새 구성 추가" 클릭하여 Smart Tomcat 클릭.

Name : myweb
Tomcat Server : 설치한 톰캣 서버 파일 로컬에서 불러오기
Deployment directory : C:\Users\admin\IdeaProjects\myweb\src\main\webapp
Context path : /myweb (일반적으로 Context 단위가 애플리케이션 단위로 보면 된다)
Server port : 8080
SSL port 가 있다면 일반적으로 8443

Apply 후 OK 클릭

우측 상단에 "myweb" 이 표시된 것을 확인할 수 있다. "myweb" 선택 후 Run 버튼 클릭하면 애플리케이션이 톰캣과 연동되어 기동된다.

http://localhost:8080/myweb/ 에 접속하면 Hello World!가 뜨는 것을 확인할 수 있다.
