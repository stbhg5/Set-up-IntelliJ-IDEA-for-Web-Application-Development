2. Automator 실행
응용 프로그램 → Automator 실행
→ "새로운 문서" 클릭
→ "폴더 적용 스크립트" 선택
→ "선택" 클릭


3. 감시할 폴더 선택
우측 상단 폴더 선택하는 부분에 스크립트를 실행할 폴더 선택


4. 셸 스크립트 추가
1) 왼쪽 검색창에 "셸 스크립트" 입력
2) "셸 스크립트 실행" 더블클릭 (오른쪽에 추가됨)
3) 상단 설정:
   - 셸: /bin/bash
   - 입력 전달: 인수로
4) 스크립트 입력:
for f in "$@"
do
    /opt/homebrew/bin/convmv -f utf8 -t utf8 --nfc --notest "$f"
done


5. 저장
파일 → 저장 (⌘S)
이름: "자소분리현상 해결 스크립트"
저장 위치: /Users/jint/Library/Workflows/Applications/Folder Actions


6. 활성화 확인
시스템 설정 → 개인정보 보호 및 보안 → 파일 및 폴더 → Automator 또는 "Folder Actions" 권한 허용
시스템 설정 → 개인정보 보호 및 보안 → 전체 디스크 접근 권한 → Automator 권한 허용

-> 완료!

- 자동으로 NFD → NFC 변환
- Windows에서 정상 표시
